Lazarus

A software tool for reconstructing ancestral molecular sequences.

Lazarus Home | Download and Install | Examples

Overview

Lazarus is a collection of scripts for running codeml and baseml in the PAML software suite. Lazarus allows you to run PAML for large distributions of phylogenies, and then integrate over those trees to calculate the maximum a posteriori ancestral sequence. Even if you're dealing with only one tree, Lazarus is still useful as it provides an easy way to compose and launch PAML jobs. Lazarus provides an easy-to-use terminal interface that mimics the program PhyML; if you're comfortable using PhyML, then using Lazarus should be straightforward.

Given a sequence alignment (in FASTA format), a phylogeny (in Newick format), and an evolutionary model (provided by PAML), Lazarus uses codeml or baseml to calculate the marginal distribution of ancestral states at all internal nodes of the given tree. After PAML is finished, Lazarus does the hard work of parsing PAML's output in order to produce a collection of text files, each containing the ancestral state distribution for a single ancestral node. For programmers, Lazarus also provides a collection of Python class objects that you can embed within your larger software pipeline. Here is some relevant peer-reviewed literature:

Maximum Likelihood Ancestral Sequence Reconstruction:
Yang et al., Genetics 1995

Marginal reconstruction:
Koshi and Goldstein, Journal of Molecular Evolution 1996

PAML, version 4:
Yang, Molecular Biology and Evolution, 2007

File Formats

Lazarus reads alignments in FASTA format and trees in Newick format. Programs such as Geneious and MacClade will help you convert your data, or you can use one of these converters.

  • Nexus to FASTA
  • Mr. Bayes *.trprobs to Newick
